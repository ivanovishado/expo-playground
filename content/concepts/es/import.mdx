# Sentencias Import

Para usar un componente como un botón o un campo de texto, primero necesitas **traerlo a tu archivo**. Para eso sirve `import`: carga las piezas que necesitas de otros paquetes.

```tsx
import { View, Text } from "react-native";
import { useState } from "react";
```

## ¿Cómo funcionan los imports?

Toda aplicación en React Native se construye a partir de piezas pequeñas y reutilizables. Con la sentencia `import`, le indicas a JavaScript cuáles piezas necesitas en tu archivo actual.

- **Imports con nombre** usan llaves: `{ View, Text }` — seleccionas exportaciones específicas del módulo.
- **Imports por defecto** no llevan llaves: `import React from "react"` — el módulo definió una exportación "principal".

## ¿Por qué importa en Expo?

Expo ofrece docenas de paquetes (`expo-camera`, `expo-location`, etc.). Cada uno se importa por separado para que tu app solo incluya lo que realmente usa, manteniéndola ligera y rápida.

```tsx
// Solo se incluye el componente Camera
import { Camera } from "expo-camera";
```

> **Tip:** Si ves un error de "module not found", seguramente necesitas instalar el paquete primero con `npx expo install`.
